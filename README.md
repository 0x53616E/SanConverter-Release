# SanConverter

### Recommended: Download through [Installer.exe](https://drive.google.com/file/d/1_tNybCjJqoNvEQKI6SxDzJk9WR5nWksA/view?usp=drive_link)

**The All-in-One Converter for StepMania to BeatUp / OTP**

SanConverter is a powerful tool designed to convert StepMania (`.sm`) chart files into Sylk (`.slk`) formats compatible with **BeatUp** (4 & 8 keys) and **One-Two Party**. It also features reversion capabilities (SLK back to SM) and a built-in analysis tool to detect chart errors.

## üì¶ Installation

We provide an automated web installer that handles everything for you.

1.  Download the **SanInstaller**.
2.  Run the executable.
3.  The installer will:
    * Download the latest version of SanConverter.
    * Set up the necessary configuration files.
    * **Add context menu entries** to Windows (Right-click support).

### Windows Context Menu
Once installed, you can simply **right-click** any `.sm` or `.slk` file to see the SanConverter menu:
* *Right-click .sm* -> **Convert to BU8 / BU4 / OTP**
* *Right-click .slk* -> **Revert or Analyze**

## üñ•Ô∏è Usage

### Graphic Interface (GUI)
1.  Open `SanConverter.exe`.
2.  Enter your **Creditor Name** (this name will appear inside the chart files).
3.  Select an option from the menu:
    * `[1]` Beat Up 8 Key - Conversion
    * `[2]` Beat Up 4 Key - Conversion
    * `[3]` One-Two Party 4 Key - Conversion
    * `[4]` One-Two Party 8 Key - Conversion
    * `[5]` Revert Beat Up .slk to .sm
    * `[6]` Revert One-Two Party .slk to .sm
    * `[7]` Analysis
    * `[0]` Console Commands (for the Converter)
4.  Select your file(s) in the file browser.
